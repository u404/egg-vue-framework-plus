# egg-vue-framework-plus

egg集成vue3.0

## Demo
[admin demo project](https://github.com/u404/admin-template--egg-vue3.0.git)

## QuickStart

```bash
$ npm install
$ npm test
```

publish your framework to npm, then change app's framework config:

```js
// {app_root}/index.js
{
  "name": "egg-showcase",
  "egg": {
    "framework": "egg-vue-framework-plus"
  }
}
```
